introduction:¶

We are going to analyze a movie dataset, that we got from TMDB, in particular I am interested in the following questions

  1. What particular factors are contributing to the popularity of movie(like budget, revenue,run time of movies and vote counts
  2. relation between popularity and revenue
  3. relation of popularity vs budget
  4. relation between popularity and vote_count
  5. Does runtime of the movie has any relationship with popularity of the movies
  6. what genre movies are made most?

# lets create function which can compare values in a series and create new values, and append those 
# values as a new column to the movie_df data frame
def category_creator(series_name, new_column_name, threshold_value):
    ''' This function compares each value in the given series with the threshold value
    and gives whether the values is more category name or less category name
    and append all the new values to a new series column and add the column
    to the movie_df dataframe'''
    new_list = []
    iterator1 = movie_df
    for x in movie_df[series_name].values:
        if x >= threshold_value:
            new_list.append('more {}'.format(series_name))
        else:
            new_list.append('less {}'.format(series_name))
    movie_df[new_column_name] = new_list

conclusion¶

Based on the above analysis we can say that, there is a corelation between popularity and budget, revenue, and vote counts of the movies generally, high budget, high revenue movies, and also high vote counts movies are more popular than low budget, low revenue and less vote count movies,

based on the graphs we can say, that vote count is one feature which differentiates very well the popularity, rather than others, like budget,revenue and runtime

there isn't a strong corelation between runtime of the movie and popularity, as the graphs are almost similar and the difference between there mean values are not that big

Drama genre movies are made most, followed by the comedy movies, looks like these two genres are most liked by directors out of all genres

limitations¶

the above conclusions and observations are not 100 percent accurate, they are correct most of the times, but can deviate in some cases, one of the reason for the limitation is missing values also many values in budget and revenue columns doesn't make sense, so they are dropped , which may alters small percent of our results
